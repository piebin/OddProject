# BGM
1)로비
인게임 전의 모든 화면에서는 <로비 BGM_1> 음악을 사용한다.
-게임을 실행하고, 메인메뉴 중 시작버튼을 누르기 전까지 실행된다.
-앨범 및 상점화면, 옵션 화면에 들어가더라도 이어서 재생된다.
-곡이 끝나면 다시 처음부터 루프된다.

2) 인게임
시작버튼을 눌렀을 때 실행되는 인게임 화면에서 BGM을 재생한다. 오픈버튼 실행 전/후를 기준으로 BGM을 적용한다.

[오픈버튼 실행 전]
-오픈 전에는 <오픈전BGM_2> 음악을 사용한다. 
-오픈 버튼 실행 시 브금 재생은 중지된다.
-브금 시간이 종료될 때까지 오픈 버튼을 누르지 않으면 다시 처음부터 루프된다.

[오픈버튼 실행 후]
-오픈 후에는 <인게임 BGM_1> 음악을 사용한다.
-오픈 버튼을 실행하고서부터 계속 재생되며, 로비 화면으로 나가기 전까지 루프된다. (위의 BGM들과 마찬가지로 곡이 끝나면 다시 처음부터 재생되는 방식)

# 삿시
삿시가 등장할 때 소리가 재생된다. 반복재생되지 않도록 한다.
-삿시가 내려갈 때 <내리는 삿시_2> 효과음이 1번만 재생된다.
-삿시가 올라갈 때 <올리는 삿시_2> 효과음이 1번만 재생된다.
-게임에서 등장하는 모든 삿시에 적용한다.

#소스 실수
떡이 꽂혀져있지 않은 상태에서 소스를 부었을 때 <소스_4_시안1+2> 효과음이 1번만 재생된다. 
-다른 상호작용을 하여도 소리가 멈추거나 끊기지 않는다. 겹출력된다. 
-다음과 같은 코멘트에도 작동한다. (떡->소스->소스(떡 건너뜀))
